# DeepSeekClient# DeepSeekClient

## 项目简介 / Project Introduction

DeepSeekClient 是一个基于 WPF 的客户端应用程序，用于与 DeepSeek API 进行交互。该应用程序允许用户发送请求并接收响应，支持流式传输和自定义 API 配置。

DeepSeekClient is a WPF-based client application for interacting with the DeepSeek API. This application allows users to send requests and receive responses, supporting streaming and custom API configurations.

## 功能特性 / Features

- 支持流式传输响应
- 自定义 API 配置
- 保存和加载对话历史
- 角色管理

- Supports streaming responses
- Custom API configuration
- Save and load conversation history
- Character management
